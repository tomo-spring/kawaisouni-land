# ホテルダンゲロウス | 大股チャレンジ

Matter.js を使った物理ベースの2D横スクロールゲーム。脚を操作して守人を避けながら風祠を目指せ。

## 起動方法

ES Modules を使用しているため、ローカルサーバー経由で開く必要があります。

```bash
# Python 3
python3 -m http.server 8080

# Node.js (npx)
npx serve .
```

ブラウザで http://localhost:8080 を開いてください。

## 操作方法

| キー | 動作 |
|------|------|
| Q | 後ろの足を前へ |
| W | 後ろの足を後ろへ |
| O | 前の足を後ろへ |
| P | 前の足を前へ |
| R | リトライ |
